Instructions

  1. Click on the map to set the start and end destination. The start is symboled with a (S) on the map, while the destination is symboled with a race flag. The form on the left hand side will automatically fill when selecting the start and destination markers

  2. A* is the chosen path finding algorithm

  3. Click Submit and you will be redirected to a page which shows:

    • --> Shortest Path Calcuated Using the Algorithm
    • --> A game Visualizer for The algorithm
